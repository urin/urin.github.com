Jekyll 1.4.2をRuby 2.0.0+Windowsに導入

Jekyll 1.4.2をRuby 2.0.0+Windowsへ導入する方法。

gem install jekyll --version="1.4.2"
gem uninstall pygments.rb
gem install pygments.rb --version="0.5.0"
gem update listen
gem install wdm

gemインストール以外についてはWindows で Jekyll 1.3 を動かすまでの手順 - てっく煮ブログにある通り。
